LiteRGSS

LiteRGSS is a kind of clone of the RGSS using SFML.

It only copies the Sprite/Viewport system and add a lot a functionalities like the shader processing on Graphics, Viewport or ShaderedSprites.

LiteRGSS render texts like Sprites using a Text object instead of Drawing them inside Bitmaps. (Bitmaps are not meant for drawing in LiteRGSS).

RubyFmod

RubyFmod is a small binding of Fmod for Ruby. It helps to access to the basic functionalities of Fmod using a single FMOD::System object.
